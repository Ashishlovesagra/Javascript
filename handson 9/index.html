Q.Find the Smaller Angle

PrepBuddy has an analog clock which consists of two hands one for hour and another for minute. She wants to calculate the shorter angle formed between hour and minute hand at any given time.
Note:You have to complete Minimal_Angle function. No need to take any input.

Input Format

The input contains two number h and m , which represents the current time as hour and minutes

Output Format

Return the Minimal angle formed between the Hour hand and Minute hand.

Constraints

All valid times

Time Limit

1 second

Example

Sample Input 1
5 30
Sample Output 1
15
Sample Input 2
6 0
Sample Output 2
180

     Code:-

    1.var Minimal_Angle = (h, m) => 
    2.{
    3.  var hour_angle = (h * 30) + (0.5 * m);
    4.  var minute_angle = (m * 6);
    5.  var angle = Math.abs(hour_angle - minute_angle);
    6.  return Math.min(360 - angle, angle);     
    7.};

Time Limit

1 second
