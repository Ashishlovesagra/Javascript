Note: As there are many shades of a color so consider integers as a color name.
i.e, color of shirt is 0,1,2, … , N.

Input Format

The first line of the input contains an integer N denoting the number of shirts in the
wardrobe.
The second line of the input contains N integers C 1 , C 2 , C 3 , C 4 , . . . , C N color of
shirts (separated by space).

Output Format

Return a single integer M denoting the unique colored shirts left in the wardrobe.

Constraints

1 <= N <= 103
1 <= C [ i ] <= 103

Time Limit

1 second

Example

Sample Input
6
3 2 4 1 2 3
Sample Output
2

Sample test case explanation

Input: C=3, 2, 4, 1, 2, 3
Output: 2
There are two 2-color shirts, and two 3-color shirts.
So, after removing 2-color and 3-color shirts, the remaining shirts are one 4-color shirt
and one 1-color shirt i.e, 4, 1.
Hence, the output will be 2.

     Code:-

    1.function Unique_Shirts (arr,N) {
    2.    var colorCount = {};
    3.    for (var i = 0; i < N; i++) {
    4.        if (colorCount[arr[i]]) {
    5.            colorCount[arr[i]]++;
    6.        } 
    7.        else {
    8.            colorCount[arr[i]] = 1;
    9.        }
    10.    }
    11.    var uniqueColors = 0;
    12.    for (var arr in colorCount) {
    13.        if (colorCount[arr] === 1) {
    14.            uniqueColors++;
    15.        }
    16.    }
    17.    return uniqueColors;
    18.}

Q.Birthday Game

1 second
