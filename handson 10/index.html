Problem 5:


Create three cards on HTML page and arrange them using flex property in row or horizontal direction and also create button at the bottom named "Change Flex direction". When user clicks on this button, the cards arrangement should be changed to vertical direction.

Code:-

        <div id="container">
        <div class="card">Card 1</div>
        <div class="card">Card 2</div>
        <div class="card">Card 3</div>
        </div>
      
        <button id="change-direction-btn">Change Flex direction</button>
      
        <script>
            var container = document.getElementById("container");
            var button = document.getElementById("change-direction-btn");
      
            button.addEventListener("click", function () {
              if (container.style.flexDirection === "row") {
                container.style.flexDirection = "column";
              } else {
                container.style.flexDirection = "row";
              }
            });
        </script>

Whatâ€™s the difference between window vs document?

Answer:-


In a web browser, the window object represents the current browser window or tab, and provides access to the browser's JavaScript environment. It contains many properties and methods that provide information about the browser and the current page, such as the location property (which provides information about the current URL), the setTimeout and setInterval methods (which can be used to schedule code to run at a later time), and the alert method (which displays a message to the user).


The document object, on the other hand, represents the current web page and provides access to the Document Object Model (DOM), which is a representation of the structure of the HTML and XML documents. It allows you to manipulate the contents of the page, such as adding or removing elements, changing the styles of elements, and listening for events. The document object contains methods such as getElementById and querySelector that can be used to access specific elements on the page, and it also has a body property that provides access to the <body> element of the page.


The window object is a parent object of the document object, so you can access document object by using window.document.


In short, window object is the global object that represents the browser window, while document is an object that represents the current web page and allows you to interact with its contents.

Problem 7:


"<h1>Hello </h1>
Add one style attribute and give text color as red and id attribute and give the id value as ""heading"" in the above given h1 tag using DOM functions"

Code:-

            <div>
            <h1>Ashish</h1>
            </div>
          
            <script>
                var heading = document.getElementsByTagName("h1")[0];
                heading.id = "heading";
                heading.style.color = "red";
            </script>

Problem 9:


Write code to implement timer clock using JS -- display HH:MM:SS -- the time should keep updating every second

Code:-

            <div id="txt"></div>

            <script>
              function startTime() {
                var today = new Date();
                var h = today.getHours();
                var m = today.getMinutes();
                var s = today.getSeconds();
                m = checkTime(m);
                s = checkTime(s);
                document.getElementById("txt").innerHTML = h + ":" + m + ":" + s;
                var t = setTimeout(startTime, 1000);
              }
              function checkTime(i) {
                if (i < 10) {
                  i = "0" + i;
                }
                return i;
              }
              startTime();
            </script>
